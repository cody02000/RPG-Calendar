Mod Information: 
For SMF 2.0.x

Creates a simple calendar for use with set dates entered under Configuration=>RPG Calendar=>Manage Events & Dates.  This page also allows adding and editing events.  They are displayed on the calendar by changing the color of the day-number and displaying a tooltip using jQuery and tipr.

Configuration=>RPG Calendar=>BBCode Calendar provides a BBCode calendar to copy into a post or private message

Adds a block for simple portal to display calendar based on current dates.

For other uses, call function rpgCal().  Automatically takes current dates, or can be supplied a start and end date (yyyy-mm-dd) [rpgCal(start,end)].
